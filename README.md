# bsBootstrap

starting suite for website projects

## develop

### init
```txt
    $ npm install
    $ grunt 
```

### develop

```txt
  $ grunt <command>

  Commands:

    default     compile all files to dist/
    dev         compile all files to dist/, create source-maps, start watcher
```
